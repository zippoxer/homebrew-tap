# Homebrew Tap

Homebrew formulas for [zippoxer](https://github.com/zippoxer) projects.

## Install

```bash
brew tap zippoxer/tap
brew install recall
```

Or directly:

```bash
brew install zippoxer/tap/recall
```

## Formulas

| Formula | Description |
|---------|-------------|
| [recall](https://github.com/zippoxer/recall) | Search and resume Claude Code and Codex CLI conversations |
